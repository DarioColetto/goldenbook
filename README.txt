GUI application Project for course PROGRAMACIÓN EN LENGUAJES ESTRUCTURADOS DE APLICACIONES DE GESTIÓN (IFCD0111).
Performs a basic CRUD.

FrontEnd:

    -Tkinter for GUI, 
    -sys & os to solve module import errors.
    -Path library to make paths available through folders

BackEnd:

    -sqlite3

Database:

    -Sqlite
    -DbBrowser(DBMS)
